## 環境建設
### 前端 (onlineJudge/frontend)
```
cd frontecn/
```
```
npm install
```
### 後端 (onlineJudge/backend)
```
cd backend/
```
1. 建立虛擬環境
   ```
   pip install virtualenv
   virtualenv venv
   ```
3. 下載環境套件
   ```
   pip install -r requirements.txt
   ```

## 執行
### 前端 (onlineJudge/frontend)
```
npm run serve
```
### 後端 (onlineJudge/backend)
```
flask run
```
